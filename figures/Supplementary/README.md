# Supplementary Figures

## Suppl. Fig. 1 — Photobleaching Control

`photobleaching_control.py` — signal-quality check across repeated baseline trials for one
animal: filtered fiber traces with fitted double-exponential photobleaching decay,
instantaneous bleaching rate, and theta-band fiber-LFP coherence over the course of a session
(Figure A), plus a day-level drift summary across recording days (Figure B).

**Inputs**: preprocessed `*_FiberPhotometry_Analysis.mat` trial files (filtered traces) and the
single-trial FieldTrip coherence outputs (`figure2_fieldtrip_trial*.mat`) from
[`../../spectral_analysis/`](../../spectral_analysis/).

**Configuration**: edit `ANIMAL`, `SESSIONS`, and `DAY_TO_SESSIONS` at the top (which sessions
belong to which recording day, and which should be combined). Paths derive from
[`../../config/paths_config.py`](../../config/paths_config.py).

**Run**:
```bash
python photobleaching_control.py
```

**Output**: `Figures/photobleaching_control/` under the project root.
